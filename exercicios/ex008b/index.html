Outras Formatações

Código Fonte

O comando document.getElementById('teste') é escrito em liguagem JavaScript (Note que na mesma fonte do resto do texto, a parte do codigo fica dificil de entender)

Com a tag "code", podemos alterar a fonte do texto para que fique mais facil de compreender, mas esta tag não organiza tambem os espaços e "enters" do codigo.
Por exemplo: num = int(input('Digite um número')) if num % 2 == 0: print('O número {num} é PAR') else: print(f'O número {num} é IMPAR') print('Fim do programa')

Para incluir os espaços e "enters" tambem, devemos utilizar a tag "pre" antes da tag "code"
Por exemplo: 

            
                num = int(input('Digite um número'))
                if num % 2 == 0:
                    print('O número {num} é PAR')
                else:
                    print(f'O número {num} é IMPAR')
                print('Fim do programa')

Citações Simples

Podemos demonstrar citações com a tag "q".
Por exemplo: Como diria um professor meu: o computador é um burro muito rápido.

Citações Completas

Para fazer citações completas, utlizamos a tag "blockquote".
Por exemplo:
Segundo Stephen Chbosky, em seu livro As vantagens de ser invisivel:

We accept the love we think we deserve.

Abreviações

Para demonstrar abreviaçoes, usamos a tag " "
Por exemplo: Estou estudando HTML e CSS. Estou curtindo muito!!

Texto Invertido

Para escrever coisas com as letras invertidas, podemos usar a tag "bdo"
Por exemplo: Estou aprendendo a criar coisas em HMTL.
