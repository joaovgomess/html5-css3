Trabalhando com listas

Listas Ordenadas

Podemos criar listas simples usando a tag "ol" e dentro dela, inserir cada item utilizando a tag "li".
Por exemplo:

  1. Acordar
  2. Tomar banho
  3. Tomar café
  4. Escovar os dentes
  5. Se arrumar
  6. Sair de casa

Podemos tambem mudar alguns componentes da nossa lista, utilizando o parametro "type" na tag "ol"
Por exemplo:

  1. Acordar
  2. Tomar banho
  3. Tomar café
  4. Escovar os dentes
  5. Se arrumar
  6. Sair de casa

Ou:

  1. Acordar
  2. Tomar banho
  3. Tomar café
  4. Escovar os dentes
  5. Se arrumar
  6. Sair de casa

Listas não Ordenadas

Tambem podemos criar listas onde a ordem não importa, e para isso usamos a tag "ul".
Por exemplo:

  • Comprar pão
  • Comprar leite
  • Comprar óleo
  • Comprar bolachas

Assim como nas listas ordenadas, na tag "ul" tambem existe o parametro "type" que serve para mudar o ícone na frente de cada item da lista.
Por exemplo:

  • Comprar pão
  • Comprar leite
  • Comprar óleo
  • Comprar bolachas

Ou:

  • Comprar pão
  • Comprar leite
  • Comprar óleo
  • Comprar bolachas

Aninhamento de listas

Podemos tambem aninhar listas dentro de outras listas. Basta colocar um "ol" ou "ul" logo abaixo de um "li".
Por exemplo:

  1. Linguagens que ja vi em 2024
    • Java
    • HTML
    • CSS
    • SQL
  2. Linguagens que ainda quero ver
    • Python
    • JavaScript
    • C#
    • C++

Lista de Definições

Tambem há a possibilidade de criarmos listas de definições, que são compostas por um termo e a definição deste termo. Para isso, usamos a tag "dl" que significa definition list, e dentro dela usamos as tags "dt" para os termos e a tag "dd" para a definição.
Por exemplo:

HTML
Linguagem de marcação para a criação do conteúdo de um site

CSS
Linguagem de marcação para a criação do design de um site

JavaScript
Linguagem de programação para a criação de interatividade de um site
